# P2P-Energy-Sharing-Project

The given repository has 3 folders.
The dark-theme folder contains the model for dark ui based website. (JavaScript)
The light-theme folder contains the model for light ui based website. (JavaScript)
The energy-p2p folder contains the ui and model for the website we showcased. (PL SQL Backend + JavaScript) 
The repository also contains the detailed report (DBMS_Report_P2P Energy Sharing.pdf) containing all the necessary information for the project.
